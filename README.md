# todo-list-v2

A full-stack Todo List application built with React, TypeScript, Node.js and Express.

## Features

- Create new todos
- Display all todos
- Update todo status
- Delete todos
- Add a deadline to each todo
- Form validation
- Loading and error handling
- Responsive design for desktop and mobile

## Validation

- Title must be at least 3 characters long
- Description is optional and can contain maximum 200 characters
- Deadline is required
- Status can be:
  - Not started
  - In progress
  - Done

## Technologies

### Frontend
- React
- TypeScript
- Vite
- CSS

### Backend
- Node.js
- Express
- JSON file storage
